# خاصية طباعة النتائج ✅

## ما تم إضافته

### 1. للإدارة (Admin)
#### الطباعة السريعة
- زر "طباعة سريعة" في صفحة `/admin/grades`
- يطبع النتائج الحالية المعروضة مباشرة
- يضيف ترويسة تلقائية بمعلومات التقرير

#### صفحة طباعة منفصلة
- رابط "صفحة طباعة منفصلة" (يظهر فقط عند اختيار قسم)
- يفتح في نافذة جديدة
- تصميم احترافي مناسب للطباعة
- يتضمن:
  - ترويسة المدرسة
  - معلومات الفلاتر (القسم، المادة، الفصل)
  - إحصائيات (عدد النتائج، المنشور، المعدل العام)
  - جدول منسق مع ألوان للعلامات
  - تذييل بتاريخ الطباعة

**الرابط:** `/admin/grades/print?classLevel=7 أساسي&classNumber=7&subject=الرياضيات`

### 2. للمعلم (Teacher)
- زر "طباعة" في صفحة `/teacher/grades`
- يطبع النتائج التي أضافها المعلم
- ترويسة تتضمن اسم المعلم
- نفس التنسيق الاحترافي

## كيفية الاستخدام

### للإدارة:
1. اذهب إلى "إدارة النتائج"
2. اختر الفلاتر (القسم، المادة، الفصل)
3. اضغط "بحث"
4. اختر إحدى الطرق:
   - **طباعة سريعة**: تطبع مباشرة من الصفحة الحالية
   - **صفحة طباعة منفصلة**: تفتح صفحة احترافية جديدة

### للمعلم:
1. اذهب إلى "النتائج"
2. اختر الفلاتر المطلوبة
3. اضغط "طباعة"

## الملفات المعدلة

### Routes
- `src/routes/admin.js` - إضافة route جديد `/admin/grades/print`

### Views
- `views/admin/grades.ejs` - إضافة زر طباعة سريعة + رابط صفحة منفصلة
- `views/admin/grades-print.ejs` - صفحة طباعة جديدة (احترافية)
- `views/teacher/grades.ejs` - إضافة زر طباعة

## مميزات التصميم

### تنسيق الطباعة
```css
@media print {
  - إخفاء الأزرار والقوائم
  - تحسين حدود الجدول
  - ألوان واضحة للعلامات
  - ترويسة وتذييل احترافي
}
```

### ألوان العلامات
- **أخضر**: 16-20 (ممتاز)
- **أزرق**: 14-15.99 (جيد)
- **برتقالي**: 10-13.99 (متوسط)
- **أحمر**: أقل من 10 (ضعيف)

## مثال على الاستخدام

```
القسم: 7 أساسي
رقم الصف: 7
المادة: الرياضيات
الفصل: الفصل الأول

النتيجة: 15 تلميذ
المعدل العام: 14.5
```
